Campaign strategy

How the campaign moves active JetBrains AI users from asking the assistant for help to delegating their first task to Junie.

From assistant to agent

Comfort with AI assistance doesn't mean a user is ready to hand a task to an agent.

The first task needs to feel small enough to try, clear enough to brief, and easy enough to verify.

Audience
JetBrains AI users active in the assistant who haven't yet completed a task with Junie.
Barrier
Asking for help is familiar; handing Junie an actual change to their project raises new questions about scope, control, and verification.
Objective
First completed Junie task within seven days of campaign entry.

HYPOTHESIS

Reduce the uncertainty around that first task — what to choose, what changes, how to verify it — and more users complete one instead of stalling at the start.

Audience thresholds, eligibility rules, and event definitions are illustrative; a real deployment would rely on internal analytics.

First-task framework

Delegating to an agent raises questions that asking an assistant never did. The framework settles one at each stage.

  • Choose

    Select one task with a clear, verifiable outcome.

    What is ready to move from discussion to execution?

  • Scope

    Set the context and the boundaries for the change.

    What should change, what must stay untouched, and how will success be judged?

  • Verify

    Read the plan, inspect the diff, and confirm the result.

    Can the user see what changed and decide confidently what to keep?

How the strategy shapes Email 1

01

Inbox promise

The subject line promises one concrete outcome; the preview shows Junie producing it. The two lines reinforce each other instead of saying the same thing twice.

02

Message role

The email introduces Junie through a single, manageable use case instead of asking users to adopt a whole new workflow.

03

Task framing and CTA

The body makes delegation feel bounded and reviewable. It shows what Junie will plan, change, test, and report back. A single primary CTA invites the user to start one task with Junie.
